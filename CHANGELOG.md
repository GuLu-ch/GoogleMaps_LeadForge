# 更新记录

本项目遵循中文更新记录。每次功能、文档、配置、数据库结构或开发流程发生变化时，都必须更新本文档。

## [未发布]

### 新增

- 初始化项目文档体系。
- 明确第一版目标：基于 Python 的 Google Maps 商家信息采集桌面软件。
- 明确技术路线：PySide6、PySide6-Fluent-Widgets、SQLite、Selenium、Playwright、JSON 配置、CSV/Excel 导出。
- 明确第一版 GUI 页面：任务配置、任务执行、结果管理、设置与文档。
- 调整 GUI 导航方式：由顶部 Tab 改为左侧侧边导航。
- 明确第一版采集字段：商家名称、地址、电话、官网、评分、评论数量、商家分类、Google Maps 链接、来源关键词。
- 明确全局去重规则：按 Google Maps 链接去重。
- 明确来源关键词合并规则：多个关键词用英文逗号分隔。
- 明确运行策略：单进程、单浏览器窗口、可视化浏览器。
- 明确暂停、继续、失败重试、连续失败自动暂停需求。
- 新增德国地区配置样例。
- 新增运行配置样例。
- 新增环境管理要求：Conda 环境名固定为 `gmap`，Python 优先使用 3.11。
- 新增依赖安装要求：pip 优先使用清华源。
- 新增驱动和自动化工具缓存目录要求：统一放入项目内 `drivers/`。
- 调整 `PySide6-Fluent-Widgets` 依赖版本为 `1.11.2`，确保当前清华 PyPI 源可安装。
- 创建项目内 Conda 环境 `.conda/gmap`。
- 安装并验证 PySide6、PySide6-Fluent-Widgets、Selenium、Playwright、pandas、openpyxl、pytest 等依赖。
- 安装 Playwright Chromium 到 `drivers/playwright-browsers/`。
- 新增 `pyproject.toml`，支持 `src/` 布局和可编辑安装。
- 新增 `pytest.ini`，配置单元测试导入路径。
- 新增配置读取模块。
- 新增关键词和 Google Maps URL 生成模块。
- 新增 SQLite 表结构初始化。
- 新增商家记录仓储，支持按 Google Maps 链接去重和来源关键词合并。
- 新增 CSV 和 Excel 导出模块。
- 新增 Selenium 和 Playwright 浏览器引擎接口骨架。
- 新增 Google Maps 列表解析器骨架，暂不写死 DOM 选择器。
- 新增 PySide6-Fluent-Widgets 左侧导航 GUI 骨架。
- 新增单元测试，覆盖配置读取、关键词生成、数据库初始化、商家去重和导出。
- 项目正式命名为 `GoogleMaps_LeadForge`。
- 新增 GPL-3.0-only 许可证。
- 新增贡献指南、行为准则、安全策略、Issue 模板和 PR 模板。
- 在 README 中新增 Git 开发流程、提交格式、回滚方式和发布前检查规范。
- 调整设置页，将浏览器、引擎、停留时间、滚动次数、失败阈值和项目路径聚合到全局设置页面。
- 明确全局运行设置与任务运行参数的关系：全局设置作为新任务默认值，任务配置页作为本次任务快照。
- 优化任务预览表、关键词队列表和结果表格的列宽策略，支持默认列宽、手动拖拽、长列拉伸和滚动条。
- 新增页面级自适应布局，支持滚动内容区、固定关键操作栏、主窗口最小尺寸和任务执行页可拖动分隔器。
- 新增 GUI 布局测试，覆盖设置页全局控件、任务参数默认值继承、表格列宽模式和页面自适应结构。
- 新增 Selenium 可视化浏览器核心能力，支持 Chrome/Edge 启动、运行时 DOM 快照、语言无关结果等待和列表滚动。
- 新增 Google Maps 列表页解析器，支持字段缺失留空和按 Google Maps 链接去重前解析。
- 新增 Google Maps 单链接采集服务，支持打开搜索链接、滚动列表、解析结果并写入 SQLite。
- 补强 `.gitignore`，忽略本地关键词、调试输出、截图、HTML、CSV、缓存、临时脚本和数据库衍生文件。
- 新增任务批次仓储，支持关键词任务持久化、状态流转、失败重试、最近可恢复批次查询和运行参数快照。
- 新增任务运行器，支持单线程顺序执行关键词、关键词间随机停留、手动暂停、停止和连续失败自动暂停。
- 新增 GUI 后台任务线程，避免采集流程阻塞 PySide6 主界面。
- 打通 GUI 从任务预览创建 SQLite 批次、进入任务执行页、启动/暂停/继续/停止、重试失败关键词、刷新结果和导出结果的流程。
- 任务配置页支持国家切换后自动重载地区复选框，并默认全选新国家地区。
- 采集服务新增页面初始停留、滚动随机停留、采集前结果 DOM 等待和商家命中关系写入。
- SQLite `task_batches` 新增 `runtime_config` 字段，并在初始化时兼容旧数据库自动补列。
- 明确当前版本实际执行采集只支持 Selenium；Playwright 保留为后续扩展入口，选择后会提示暂不支持。
- 优化任务执行页状态反馈，展示启动中、运行状态、当前关键词、国家、地区、城市、浏览器引擎和商家统计。
- 新增 `scripts.open_login_browser`，用于打开与采集任务共用用户目录的 Google 登录浏览器。
- 新增 `scripts.cleanup_runtime_data`，用于清理本地运行数据库、日志、导出和调试输出，同时保留关键词输入和浏览器登录缓存。

### 说明

- `source_keywords` 字段只保存用户输入的行业关键词，完整搜索词由命中关系表保存，避免逗号分隔歧义。

### 暂不支持

- 街道级搜索。
- 多浏览器并发。
- 无头模式。
- 自动绕过验证码或风控页面。
- 点击商家详情页深度采集。
- 云端同步。
